Programming Manual: Inline Perl Sample 2

In version 1.4, "primitives" were included to allow inline Perl documents to include entire files easily. For example, this block:

<perl>&finclude("/usr/local/htdocs/httpi/license.txt")</perl>

inserts that file into this document, like so (obviously the table tags aren't part of it). If you have this sample inline Perl file installed locally, the directive will probably fail (but check the source for the error message, or you'll miss it).

&finclude("/usr/local/htdocs/httpi/license.txt")
